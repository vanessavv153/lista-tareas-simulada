# Lista de Tareas - Proyecto de Aprendizaje Git

## Descripción
Proyecto simulado para aprender flujo de trabajo colaborativo con Git/GitHub.

## Funcionalidades Implementadas
- ✅ Agregar nuevas tareas
- ✅ Eliminar tareas existentes  
- ✅ Diseño responsive con CSS

## Proceso de Desarrollo

### Roles Simulados
1. **Integrante 1:** Funcionalidad agregar tareas (rama )
2. **Integrante 2:** Funcionalidad eliminar tareas (rama )  
3. **Integrante 3:** Diseño de interfaz (rama )

### Flujo de Trabajo
1. Creación de ramas independientes por funcionalidad
2. Commits descriptivos en cada rama
3. Pull Requests y revisión de código
4. Resolución de conflictos de fusión
5. Integración a rama principal (main)
